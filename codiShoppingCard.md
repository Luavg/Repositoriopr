
# GitLab

GitLab es una plataforma para alojar repositorios Git y trabajar con control de versiones.  
Incluye funciones para gestionar código, issues, integración continua (CI/CD) y colaboración en equipo.


# Pull Request

## ¿Qué es una Pull Request?
Una Pull Request es una petición para mezclar los cambios de una rama de trabajo en otra rama, normalmente en la rama `main`.  
Sirve para proponer cambios y que otra persona los revise antes de añadirlos al proyecto principal.

## Para qué sirve
- Permite revisar el código antes de fusionarlo.
- Muestra claramente qué archivos y líneas han cambiado.
- Facilita que varias personas trabajen en el mismo proyecto sin pisarse.

## Ventajas
- Mejora la calidad
